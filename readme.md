
# Truelogic Java Assessment

Assessment to apply for Java Developer Role



## Requirements
Java17, Maven 3.0.4
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  mvn clean install
```

Run the application

```bash
  mvn exec:java
```
